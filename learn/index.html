Questions

Frequently asked questions

What is the Learning Hub?

It's a single place that organizes the books, articles, tools, and services already on this site by topic and by learning path. It doesn't add new content of its own.

Do I need to follow a Learning Path in order?

No, each path is a suggested order, not a requirement. Jump to whatever's useful for your situation.

Is anything on this page paid?

Only the eBook in the Investing section has a price. Every article, calculator, the Goal Planner, and a service enquiry are free to explore.

How is this different from the Goal Planner?

The Goal Planner asks about one specific goal and calculates a suggested savings figure. The Learning Hub is a directory: it points you to everything relevant, including the Goal Planner itself.

Where should I start if I'm completely new?

Try the Financial Basics section above, or the Start Investing and Build an Emergency Fund learning paths.

Will search be added to this page?

Not yet. The page is built so a search or filter tool can be added later without restructuring it. For now, browse by path or by topic.
